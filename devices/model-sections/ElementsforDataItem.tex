\label{sec:Source}

Descriptions for attributes of \block{Source}:

\begin{itemize}

\item \property{componentId}[Source] : The identifier attribute of the \block{Component} element that represents the physical part of a piece of equipment where the data represented by the \block{DataItem} element originated.

\item \property{compositionId}[Source] : The identifier attribute of the \block{Composition} element that the reported data is most closely associated.

\item \property{dataItemId}[Source] : The identifier attribute of the \block{DataItem} that represents the originally measured value of the data referenced by this data item.

\item \property{value}[Source] : The value of the id of the source element.
\end{itemize}
